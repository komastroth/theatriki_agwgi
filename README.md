# theatriki_agwgi
Ψηφιακό μαθησιακό αντικείμενο για το γνωστικό αντικείμενο της θεατρικής αγωγής

## Τίτλος μαθησιακού αντικειμένου
Μαθαίνω τα πιο γνωστά αρχαία θέατρα της Ελλάδας

## Περιγραφή
Το ψηφιακό μαθησιακό αντικείμενο είναι ένα διαδραδτσικό βιβλίο (interactive book) που έχει δημιουργηθεί με H5P μέσα από την πλατφόρμα η-τάξη (https://eclass.sch.gr/) για την γνωριμία και την εξοικείωση παιδιών ηλικίας 10-12 ετών με τα πιο γνωστά αρχαία θέατρα της Ελλάδας. Αυτό το αντικείμενο περιλαμβάνει interactive hotspots σε ένα χάρτη της Ελλάδας με πληροφορίες για γνωστά αρχαία θέατρα και ένα quiz για την ενίσχυση της μαθησιακής διαδικασίας.

## Προέλευση Υλικού
Αυτό το μαθησιακό αντικείμενο αναπτύχθηκε αρχικά για χρήση στην πλατφόρμα e-class για την ενίσχυση της εκπαιδευτικής διαδικασίας. Σκοπός είναι να διατεθεί ελεύθερα για εκπαιδευτικούς και μαθητές μέσω του GitHub.

## Σκοπός
Το αντικείμενο αυτό δημιουργήθηκε για να βοηθήσει τα παιδιά να κατανοήσουν και να εκτιμήσουν την αρχαία ελληνική θεατρική τέχνη και αρχιτεκτονική μέσα από διαδραστικά εκπαιδευτικά εργαλεία.
